Install
-------
- #### Maven
- #### java1.8+
- #### spread-lang
``` xml
<dependency>
  <groupId>club.spreadme</groupId>
   <artifactId>spreadme-basic</artifactId>
  <version>1.5.0</version>
</dependency>
```
